# ofxGgmlRag v1.0.1

## Compatibility

- openFrameworks: `of_v20260406_vs_64_release`
- minimum `ofxGgmlCore`: `1.0.1` / commit `2171e8b`
- `OFXGGML_RAG_VERSION_STRING`: `1.0.1`

## Highlights

- Added independent RAG addon version metadata.
- Exposed version metadata through `ofxGgmlRag.h`.
- Documented the release checklist and release policy for RAG workflows.
- Kept RAG-specific request helpers out of `ofxGgmlCore`.
- Preserved the root-level `ofxGgmlRagSearchExample` as the first example lane.

## Runtime Scope

This release includes public request/result helper types, input validation,
description helpers, a root-level citation search example skeleton, local
validation, and headless tests.

It does not include complete document ingestion, web crawling, chunking,
embedding generation, vector search, citation extraction, or project memory
runtime adapters yet. Those remain roadmap tasks.
